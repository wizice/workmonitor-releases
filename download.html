설치 후 설정

  1. input 그룹 추가 (키보드/마우스 수집에 필요)
    sudo usermod -aG input $USER → 재로그인
  2. 설정 파일 편집 (Cloud Sync 사용 시)
    nano ~/.local/share/WorkMonitor/cloud_sync_lite.json
    worker_url, api_key 설정 → enabled: true
  3. 서비스 시작 및 자동실행 등록
    systemctl --user enable --now work-monitor
  4. 상태 확인
    systemctl --user status work-monitor
    journalctl --user -u work-monitor -f

Claude Code로 자동 설치

다른 리눅스 PC에서 Claude Code가 설치를 도와줍니다. Claude Code 터미널에 아래를 입력하세요:

# Claude Code에 입력 https://wizice.github.io/workmonitor-releases/install-spec.json 을 읽고 work-monitor를 이 리눅스 PC에 설치해줘

Claude Code가 설치 스펙(JSON)을 읽어 OS 환경에 맞게 단계별 설치를 진행합니다.

동작 방식

  1. 스펙 읽기 — Claude Code가 install-spec.json을 WebFetch로 다운로드
  2. 환경 판단 — OS, 아키텍처를 확인하여 맞는 바이너리 URL 선택
  3. 설치 실행 — install_steps를 순서대로 실행 (바이너리, input 그룹, 설정, systemd)
  4. 문제 해결 — 에러 발생 시 troubleshooting 섹션 참조하여 자동 대응

Claude Code가 아직 없다면

# Claude Code 설치 (macOS / Linux / WSL) curl -fsSL https://claude.ai/install.sh | bash

Work Monitor란?

PC 활동(키보드/마우스)을 모니터링하여 근무 시간과 집중도를 추적하는 데몬 프로그램입니다.

  • 1분 간격 이벤트 수집, 10분 간격 클라우드 동기화
  • Claude 세션 로그 분석 및 AI 코칭
  • 텔레그램 봇 연동 (원격 조회/제어)
  • 자동 업그레이드 (시작 시 버전 체크)

v1.5.1 변경사항

  • Linux headless 빌드 지원 (Tauri 의존 분리)
  • TLS 백엔드 rustls 전환 (libssl 의존 제거)
  • Linux 자동 업그레이드 지원
  • systemd 사용자 서비스 설치 스크립트
  • OS별 버전 매니페스트 자동 선택
